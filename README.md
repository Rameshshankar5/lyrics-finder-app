\# 🎵 Lyrics Dreamer App



A full-stack mobile application built using React Native and Node.js that allows users to search and view song lyrics.



\## 🚀 Features

\- Search song lyrics by title and artist

\- Clean glass UI design

\- Fast API-based search

\- Cross-platform (Android \& iOS)



\## 🛠️ Tech Stack

\- React Native (Expo)

\- Node.js + Express

\- LRCLIB API

\- Axios



\## 📱 Screens

\- Search screen

\- Lyrics result screen



\## ⚙️ Setup



\### Backend

cd backend

npm install

node server.js



\### Mobile

cd mobile

npm install

npx expo start
